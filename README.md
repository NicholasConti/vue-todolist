TODO LIST
- app che mostra una todo list
- al click dell icona del cestino il programma elimina l elemento della lista
- se inserisco un todo e premo il bottone + aggiungo il todo alla lista
    - aggiunto il controllo se l utente non inserisce nulla e preme il bottone
    - aggiunta possibilita di aggoingere il todo premendo il tasto invio della tastiera
-  se clicco sul todo il testo viene sbarrato e il valore done del todo diventa true
